# Circle Time

Meeting-room booking and management platform.

## Structure

```
circle-time/
├── backend/   — API server (planned)
├── web/       — Vite + React web dashboard
├── mobile/    — React Native panel app
└── README.md
```

## Web (`web/`)

Vite + React single-page application for room search, booking, and admin analytics.

```bash
cd web
npm install
npm run dev
```

## Mobile (`mobile/`)

React Native app for in-room panel displays (check-in, meeting status, end-early).

```bash
cd mobile
npm install
npx react-native run-android
```
